Part 2: Making Decisions with Conditional Statements

Conditional statements allow your program to make decisions based on conditions. Think of it like real life: "If it rains, take an umbrella. Otherwise, don't."

1. The if Statement

The if statement executes code only when a condition is True.

if_statement.py
# Simple if statement
age = 20

if age >= 18:
    print("You are an adult")
    print("You can vote")

print("Program continues...")
Output:
You are an adult
You can vote
Program continues...

2. The if-else Statement

The if-else statement provides an alternative action when the condition is False.

if_else.py
marks = 45

if marks >= 50:
    print("Congratulations! You passed.")
else:
    print("Sorry, you failed.")
    print("Better luck next time!")

print("Exam result displayed")
Output:
Sorry, you failed.
Better luck next time!
Exam result displayed

3. The elif Statement

The elif (else if) statement allows you to check multiple conditions in sequence.

elif_statement.py
score = 75

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")
Output:
Grade: C
Important Points about elif:
  • Python checks conditions from top to bottom
  • Once a condition is True, it executes that block and skips the rest
  • You can have as many elif statements as needed
  • The else statement is optional

4. Complete Example: Student Grading System

Project: Automatic Grade Calculator

# Student Grading System
student_name = "Rahul"
marks = 78

print("====== Student Report Card ======")
print(f"Student Name: {student_name}")
print(f"Marks Obtained: {marks}")
print("=================================")

# Determine grade
if marks >= 90:
    grade = "A+"
    remark = "Outstanding!"
elif marks >= 80:
    grade = "A"
    remark = "Excellent!"
elif marks >= 70:
    grade = "B"
    remark = "Very Good!"
elif marks >= 60:
    grade = "C"
    remark = "Good"
elif marks >= 50:
    grade = "D"
    remark = "Needs Improvement"
else:
    grade = "F"
    remark = "Failed - Work Harder"

print(f"Grade: {grade}")
print(f"Remark: {remark}")

# Check pass/fail status
if marks >= 50:
    print("Status: PASS")
else:
    print("Status: FAIL")
